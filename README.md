## Youtube Clone Coding

1. 프로젝트 설명
 - Youtube API를 사용하여 동영상 검색, 재생 기능을 구현한다.
 - Main 화면에는 상위 24개의 비디오를 보여준다.
 - 검색 창을 이용하여 비디오를 검색할 수 있다.
 - 비디오를 클릭하면 비디오가 재생되고, 채널이름, 설명 등등을 볼 수 있다.
